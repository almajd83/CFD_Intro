# CFD with MATLAB
## Introduction to PDEs and Navier–Stokes Equations

This repository contains teaching material for an introductory course in Computational Fluid Dynamics (CFD) using MATLAB.

The course develops numerical methods for partial differential equations (PDEs), progressing from simple one-dimensional models to the incompressible Navier–Stokes equations and an introduction to compressible flow.

---

## Repository Structure

CFD-MATLAB-Intro-to-Navier-Stokes/

- Lectures/  
  Theory notes covering the mathematical and physical foundations  

- Workbook/  
  Problem set used throughout the course  

- Matlab_code/  
  MATLAB implementations corresponding to each workbook problem

- Textbook/
  This textbook forms the core of the course, providing theoretical interpretations and MATLAB implementations of numerical methods for partial differential equations and Computational Fluid Dynamics.

---

## Course Content

The course follows a structured progression:

1. Linear convection (1D)  
2. Nonlinear convection  
3. Diffusion  
4. Burgers’ equation  
5. Extension to two dimensions  
6. Laplace and Poisson equations  
7. Incompressible Navier–Stokes equations  
8. Compressible flow and Euler equations  

Each problem introduces a specific concept and contributes to the development of a CFD solver.

---

## How to Use

The recommended workflow is:

1. Read the relevant lecture notes in the Lectures folder  
2. Attempt the corresponding problem in the workbook  
3. Use the MATLAB script in Matlab_code for implementation and verification
4. The textbook provides the mathematical background and a walkthrough the MATLAB codes.

Each MATLAB file corresponds directly to a problem in the workbook and is organised sequentially to match the course progression.

Users are encouraged to review, test, and adapt the code as part of the learning process.
